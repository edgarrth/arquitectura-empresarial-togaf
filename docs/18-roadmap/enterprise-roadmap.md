# Enterprise Roadmap

# North Star

Arquitectura financiera modular, segura, gobernada y observable, orientada a capacidades de negocio y preparada para innovación digital.

# Horizonte 0-3 meses

- Formalizar Architecture Board.
- Publicar principios de arquitectura.
- Crear inventario inicial de aplicaciones.
- Definir estándares API/eventos.
- Implementar observabilidad mínima en servicios críticos.
- Seleccionar estrategia CIAM.

# Horizonte 3-6 meses

- Implementar API Management.
- Implementar CIAM base.
- Crear catálogo de APIs y eventos.
- Definir dominios de datos y owners.
- Crear golden path de microservicios.
- Iniciar event platform.

# Horizonte 6-12 meses

- Modernizar dominios cliente, pagos y comercios.
- Implementar data governance operativo.
- Automatizar conciliación prioritaria.
- Implementar SLOs por producto.
- Integrar policy as code.
- Reducir integraciones punto a punto.

# Horizonte 12-18 meses

- Expandir plataforma como producto.
- Desacoplar capacidades críticas del core.
- Consolidar data products.
- Implementar decisioning moderno de riesgo/fraude.
- Medir reducción de deuda técnica y costo operativo.
